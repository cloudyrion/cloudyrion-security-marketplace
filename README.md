# CLOUDYRION Marketplace

Plugin marketplace for Claude Code by CLOUDYRION.

## Available Plugins

- **[cloudyrion-security](plugins/cloudyrion-security/)** — Comprehensive security assessment suite

## Usage

Add this marketplace to Claude Code:

```shell
/plugin marketplace add cloudyrion/cloudyrion-security-marketplace
```

Then install plugins:

```shell
/plugin install cloudyrion-security@cloudyrion-marketplace
```
